# PerekupAlfa v1.0 (console, rus_lang)
The application was created to calculate the cost of cars and get benefits when purchasing on the Grand Mobile project.

Main menu:

=============================================
Введите номер команды:
1 - Показать список авто
2 - Расчет выгоды и рекомендация по покупке
3 - Вывод информации по ID
0 - Выход
=============================================

1. The item is responsible for viewing a list of cars indicating their ID, name and price (in the showroom).
2. The item is responsible for calculating the 50% and 75% cost of the car, and also offers advice.
3. The item is responsible for displaying information about the car (ID, name, price (in the showroom)) when you enter its ID.
4. The item is responsible for completing the program.
